Desafio Back-End Express

Projeto de desenvolvimento pessoal onde criei uma API para gerenciamento de livros. A API permite adicionar, editar, excluir e listar livros, além de marcar se um livro foi lido ou não. Essa solução foi projetada para facilitar o acompanhamento de leituras, proporcionando flexibilidade e praticidade na organização de livros lidos e não lidos.
